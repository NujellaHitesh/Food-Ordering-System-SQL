# 🍔 Online Food Ordering System (SQL Project)

## 📌 Project Overview
This project simulates a real-world **Online Food Ordering System** using MySQL.  
It includes database design, data insertion, and 250+ SQL queries ranging from basic to advanced.

---

## 🗂️ Database Structure
- Users
- Restaurants
- Menu
- Orders
- Order_Items

---

## 🚀 Features Covered
- Basic Queries (SELECT, WHERE, LIKE, BETWEEN)
- Joins (INNER, LEFT, Multi-table)
- Aggregations (COUNT, SUM, AVG, GROUP BY, HAVING)
- Subqueries (Nested, Correlated)
- Window Functions (RANK, ROW_NUMBER, LAG, LEAD)
- Advanced SQL (CTE, CASE, Performance)
